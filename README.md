# expense_tracker_project
A lightweight Python console app to record and manage daily expenses. Users can add entries with amount, category, and date, view all logged expenses, and see category-wise summaries. Inputs are validated for accuracy, and all data is saved securely in a JSON file for easy retrieval and tracking.


# Expense Tracker (Console App)

A simple Python-based console application to track expenses.  
Features:
- Add new expenses (amount, category, date)
- View all expenses
- Summary by category
- Validates input and stores data in JSON file

## Run the app
```bash
python main.py
```
